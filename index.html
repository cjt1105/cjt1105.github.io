Streambuddies
MEAN Stack

Streambuddies is a MEAN stack app that allows users to split up their streaming subscriptions with their roommates. Users log on using Facebook OAuth and are able to add the login info for their streaming accounts. Users can then invite friends to join through a unique invite link. By joining the account, the user agrees to split the bill. Using webhooks, the subscription price is dynamically altered each month so that each user pays an equal portion. All payment processing is handled through the Stripe Connect API.
